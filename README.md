A Chrome extension that translates Chinese text on any webpage. Select Chinese text to see its English translation and pinyin in a tooltip.

## Install

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode** (toggle in the top-right corner)
4. Click **Load unpacked** and select the `chinese_lens` folder
5. The extension is now active on all pages â€” select any Chinese text to see the translation

## How It Works

Highlight Chinese text on any webpage and a tooltip appears showing:

- The original Chinese characters
- Pinyin romanization
- English translation

Translation is powered by Google Translate (no API key needed).
